Part 2 (Data cleaning using dplyr)

In this exercise, we will calculate the rate of inflation for our assigned country. It is useful to have a look at lab_4.R which is available on Blackboard. The growth rate of the price index \(P_{t}\) between month \(t\) and \(t-12\) is given by \[\begin{equation} \pi_{t} = 100\times\frac{P_{t} - P_{t-12}}{P_{t-12}}. \end{equation}\] The slides for this week might be useful for this exercise. You can find these here here.

Some dplyr functions

mutate(): create new column or manipulate existing column.

select(): select columns to keep in the dataframe.

rename(): rename column.

filter(): select rows in a dataframe.

group_by(): perform some subsequent operation by group.
